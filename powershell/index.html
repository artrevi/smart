Useful Powershell Commands


Alter Brightness : Where x = 1 - 100, (Get-WmiObject -Namespace root/WMI -Class WmiMonitorBrightnessMethods).WmiSetBrightness(1,x)


Volume Up : (new-object -com wscript.shell).SendKeys([char]174)

Volume Up : (new-object -com wscript.shell).SendKeys([char]175)

Toggle Mute : (new-object -com wscript.shell).SendKeys([char]173)


Download Speed: function Measure-NetworkSpeed { $TestFile = 'https://ftp.sunet.se/mirror/parrotsec.org/parrot/misc/10MB.bin'; $TempFile = Join-Path -Path $env:TEMP -ChildPath 'testfile.tmp'; $SpeedMbps = (10 / (Measure-Command { (New-Object Net.WebClient).DownloadFile($TestFile, $TempFile) }).TotalSeconds) * 8; "{0:N2} Mbit/sec" -f $SpeedMbps } Measure-NetworkSpeed
